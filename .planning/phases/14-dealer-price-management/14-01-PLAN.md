# Phase 14 Plan 01: Dealer Modal Redesign — Split-Panel Catalog Layout

## Objective

Refactor `DealerManagementModal.tsx` from a single-column expand-in-place form list into a two-zone split-panel layout: a dealer sidebar on the left and a metal-price catalog panel on the right. Inspired by the Holland Gold buy/sell price interface. All existing CRUD functionality (create, edit, delete) is preserved — only the layout and visual treatment change.

## Context

- Existing modal: `max-w-xl`, single column, each dealer row expands inline to show a form
- Current pattern is functional but cramped and form-heavy — every dealer always shows 6 price fields
- Research: see `14-RESEARCH.md` — split-panel catalog pattern, no new libraries needed
- Dealer + price API: unchanged (`api.dealers.*` from Phase 13)
- Design tokens: `--color-primary` (#e9c349), `--color-surface-high`, `--color-outline-variant`
- Heading font: `font-headline italic` (Newsreader), body: Inter
