Append three new endpoints to `portfolioRouter` in `api/src/routes/portfolio.ts`, after the existing `/summary` handler.

**GET /api/portfolio/threshold**
Read `concentration_threshold` from `_schema_info`. If key doesn't exist, return `{ threshold: 70 }` as default (do not insert). Example:
```ts
portfolioRouter.get('/threshold', async (_req, res, next) => {
  try {
    const row = await db('_schema_info').where({ key: 'concentration_threshold' }).first();
    const threshold = row ? Number(row.value) : 70;
    res.json({ threshold });
  } catch (err) { next(err); }
});
```

**PUT /api/portfolio/threshold**
Accept body `{ threshold: number }`. Validate: must be integer 1–99 (inclusive). Upsert into `_schema_info`. Return `{ threshold: number }`. Example:
```ts
portfolioRouter.put('/threshold', async (req, res, next) => {
  try {
    const val = Number(req.body?.threshold);
    if (!Number.isInteger(val) || val < 1 || val > 99) {
      res.status(400).json({ error: { message: 'threshold must be an integer 1–99', status: 400 } });
      return;
    }
    await db('_schema_info')
      .insert({ key: 'concentration_threshold', value: String(val) })
      .onConflict('key').merge({ value: String(val) });
    res.json({ threshold: val });
  } catch (err) { next(err); }
});
```

**GET /api/portfolio/warnings**
1. Read threshold from `_schema_info` (default 70 if missing)
2. Compute total portfolio value from `assets` table
3. JOIN `tags → asset_tags → assets`, GROUP BY `tags.id`, HAVING `portfolio_pct >= threshold`
4. For each tag over threshold, fetch contributing assets sorted by value DESC
5. Compute `severity`: `portfolio_pct >= 90` → `'red'`, else → `'amber'`
6. Sort results: red first, then by `portfolio_pct DESC`

Response shape:
```ts
[{
  tag_id: string,
  tag_name: string,
  portfolio_pct: number,    // rounded to 2 decimal places
  combined_value: number,
  asset_count: number,
  severity: 'amber' | 'red',
  assets: [{ id: string, name: string, value: number, pct: number }]
}]
```

Use a single query with `JSON_AGG` for assets to avoid N+1:
```ts
const rows = await db('tags as t')
  .join('asset_tags as at', 'at.tag_id', 't.id')
  .join('assets as a', 'a.id', 'at.asset_id')
  .select(
    't.id as tag_id',
    't.name as tag_name',
    db.raw('COUNT(DISTINCT a.id)::int AS asset_count'),
    db.raw('SUM(a.value)::float AS combined_value'),
    db.raw(`
      JSON_AGG(
        JSON_BUILD_OBJECT('id', a.id, 'name', a.name, 'value', a.value::float)
        ORDER BY a.value DESC
      ) AS assets_json
    `),
  )
  .groupBy('t.id', 't.name');

// Then compute portfolio_pct using total_value (separate query), filter, sort
```

When `total_value` is 0 or there are no tags, return `[]`.
